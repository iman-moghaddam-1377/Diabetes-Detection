# Diabetes Prediction Using Machine Learning

## Overview

This project focuses on predicting diabetes using machine learning techniques. It analyzes patient health-related features from a diabetes dataset to determine whether a patient is likely to have diabetes.

The objective is to build accurate predictive models that can assist in the early detection of diabetes based on medical measurements.

---

## Features

* **Diabetes Prediction**

  * Predicts whether a patient has diabetes using the features available in the dataset.

* **Data Preprocessing**

  * Includes data cleaning, handling missing values, feature scaling, and exploratory data analysis (EDA).

* **Multiple Machine Learning Models**

  * The dataset is trained and evaluated using **seven different machine learning algorithms** to compare their performance.

* **Model Evaluation**

  * Each model is evaluated using standard classification metrics, and their performances are compared to identify the best-performing model.

* **High Prediction Accuracy**

  * The best-performing model achieved an accuracy of **90%** on the dataset.

---

## Machine Learning Models

The project evaluates the following machine learning algorithms:

* Logistic Regression
* K-Nearest Neighbors (KNN)
* Support Vector Machine (SVM)
* Decision Tree Classifier
* Random Forest Classifier
* Gradient Boosting Classifier
* XGBoost

---

## Project Structure

All project code is contained in a single Jupyter Notebook:

```text
main.ipynb
```

The notebook includes:

* Data loading
* Exploratory Data Analysis (EDA)
* Data preprocessing
* Feature engineering
* Model training
* Model evaluation
* Performance comparison
* Final results

---

## Results

After training and comparing all seven machine learning models, the highest classification accuracy achieved was **90%**. The notebook also includes performance metrics and comparisons to help evaluate each model's effectiveness.
